# Framework-SIM
Automação em Python para engenharia de dados do SUS. Consome dados do Sistema de Informações sobre Mortalidade (SIM) via API , filtrando para Araranguá/SC. Realiza limpeza, anonimização (LGPD) e traduz códigos (CID-10 e idade) para linguagem cidadã. Gera tabelas estruturadas e gráficos estatísticos locais.
